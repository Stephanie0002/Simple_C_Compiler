# Simple_C_Compiler

**2.文件夹结构**

/viewTree ---存放一个python项目，用于可视化语法分析树，需要电脑上安装有Graphvix方可显示出图片

linux：bash run.bash              (进行语法分析，输出语法树文件)
       python ./viewTree/main.py  (可视化刚刚生成的语法树)
